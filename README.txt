Plugin for tipping authors in various cryptocurrencies.

== Description ==

Crypto tip is a plugin to allow users to tip content creators in various cryptocurrencies. Currently supporting Karma Coin. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to plugins in your admin control panel and select Add New
2. Select upload and browse for "cryptotip.zip" then select install
3. Activate the cryptotip plugin
4. Go to crypto tips settings and configure the rpc url for your cryptocurrency wallet. A wallet with rpc set up is a must to use this plugin.

== Frequently Asked Questions ==

= How does it work? =

At the bottom of every blog post a coin address will be displayed for whatever currency you choose to receive tips in along with the total amount you've received in tips.
